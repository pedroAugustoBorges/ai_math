# Math for ML/AI: A Learning Journey

This repository is a personal documentation of my journey through the mathematical foundations of Machine Learning and Artificial Intelligence. 

The goal is to implement every core concept from scratch in Python to build a deep intuition, then compare them with industry-standard library implementations (NumPy, SciPy, etc.).

## 🗺️ Roadmap & Progress

### 🟢 Phase 1: Statistics (Completed)
- [x] Mean, Variance, and Covariance
- [x] Correlation and Standard Deviation

### 🟡 Phase 2: Probability & Distributions (In Progress)
- [x] Basic Probability Concepts
- [ ] Conditional Probability & Bayes' Theorem
- [ ] Law of Large Numbers (LLN)
- [ ] Distributions: Normal, Uniform, Binomial
- [ ] Likelihood & Maximum Likelihood Estimation (MLE)

### 🔴 Phase 3: Linear Algebra
- [ ] Vectors, Matrices, Dot Product, and Norms
- [ ] Eigenvalues & Eigenvectors
- [ ] Singular Value Decomposition (SVD) & PCA
- [ ] Vector Spaces, Basis, and Rank

### 🔴 Phase 4: Calculus & Optimization
- [x] Basic Derivatives
- [x] Chain Rule
- [ ] Gradients & Optimization
- [ ] Jacobian & Hessian Matrices
